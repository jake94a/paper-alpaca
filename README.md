# paper-alpaca

This project requires:
- Quiverquant account
    - Quiverquant api key
- Alpaca Paper Trading Account
    - Alpaca API_KEY_ID
    - Alpaca API_SECRET_KEY
- .env file containing
    - APCA_API_BASE_URL=https://paper-api.alpaca.markets
    - APCA_API_KEY_ID=`REDACTED`
    - APCA_API_SECRET_KEY=`REDACTED`
    - QUIVER_API_TOKEN=`REDACTED`
